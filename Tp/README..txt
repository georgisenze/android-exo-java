ENONCER DE L'EXERCICE : 
                        Dans cet exercice nous avons créé une classe mère abstraite 'Employe' qui contient les
                        attributs 'nom' pour le nom de l'employer de et 'poste' pour le poste de l'employer, et une
                        methode abstraite 'Duree' pour donner le durée de travail de l'employer.
                        Par suite nous avons créé deux sous-classe 'EmployeSalarie' et 'EmployeTemporaire'.

                            -> 'EmployeSalarie' prend en compte les employer à plein temps, nous ajoutons deux attributs
                            'salaireFixe' et 'tauxActivite' 

                            -> 'EmployeTemporaire' prend en compte les employer temporaire, et nous ajoutons deux attributs
                            'salaireHoraire' et 'heures'

                          Une interface 'merite' contenant une methode 'securiteSocial' implementé dans la classe des EmployeSalarie.
